# My XFCE Picom Dotfiles

My picom dotfiles for XFCE, with blur and rounded corners

# Install Picom:

openSUSE:
```
sudo zypper in picom
```
Arch:
```
sudo pacman -S picom

```
Debian:

```
sudo apt install picom
```
Ubuntu doesn't has it in the repositories, so go build it yourself
